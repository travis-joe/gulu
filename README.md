## 使用
1.添加css样式
使用前，在css中开启 border-box
```
*, *::before, *::after{box-sizing: border-box;}
```
设置css变量
```
html {
    --button-height: 32px;
    --font-size: 14px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 4px;
    --color: #333;
    --border-color: #999;
    --border-color-hover: #666;
}
```
2.安装
```
npm i --save gulu-ui-test
```
